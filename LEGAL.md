# Legal & Clean-Room Declaration

This project is a **clean-room implementation** of the Socket.IO v4 protocol for Unity.

## Clean-Room Statement

- No source code from the official Socket.IO JavaScript client has been referenced.
- No source code from any paid or closed-source Unity Socket.IO asset has been referenced.
- No internal implementation details from other clients have been copied or mirrored.

This implementation is based solely on:
- Publicly available Socket.IO and Engine.IO protocol documentation
- Observed network behavior using black-box testing
- Original design and independently written source code

## Trademark Notice

"Socket.IO" is a trademark of its respective owners.

This project is **not affiliated with, endorsed by, or sponsored by Socket.IO**.

## Intent

The intent of this project is to provide an open, transparent, and community-driven
Socket.IO v4 client for Unity developers while fully respecting intellectual property
and licensing boundaries.
